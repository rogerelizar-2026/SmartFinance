1. Visão Geral do Painel (Dashboard)

O Dashboard é o coração da aplicação SmartFinance, onde você terá uma visão consolidada de toda sua situação financeira em tempo real.

🔍 Elementos do Dashboard

Saldo Total: Mostra o saldo combinado de todas suas contas ativas, atualizado automaticamente conforme novas transações são registradas.

Gráfico de Balanço Mensal: Exibe visualmente a relação entre receitas e despesas ao longo dos últimos 12 meses, permitindo identificar padrões sazonais.

Cartões de Alerta: Notificações inteligentes sobre contas próximas do vencimento, metas financeiras atingidas ou possíveis problemas de fluxo de caixa.

Dica Pro: Personalize o período de visualização no canto superior direito para analisar dados específicos por ano, trimestre ou mês.

📈 Navegação Intuitiva

A interface segue os princípios de design flat e material, com ícones intuitivos e cores que representam diferentes categorias financeiras:

  • Verde: Receitas e saldos positivos
  • Vermelho: Despesas e alertas
  • Azul: Investimentos e metas
  • Laranja: Cartões de crédito

2. Primeiros Passos: Contas e Cartões

A configuração inicial das contas é fundamental para o funcionamento eficiente do sistema SmartFinance.

💳 Adicionando Contas Bancárias

Clique em "Contas" no menu lateral e selecione "Adicionar Nova Conta". Preencha os seguintes campos:

- Nome da Conta (ex: "Corrente Itaú", "Poupança Caixa")
- Tipo de Conta (Corrente, Poupança, Investimento)
- Saldo Inicial
- Banco e Agência (opcional para referência)
- Limite Disponível (para contas com cheque especial)

💳 Gerenciando Cartões de Crédito

Para adicionar cartões de crédito:

  1. Clique em "Cartões" no menu
  2. Selecione "Novo Cartão"
  3. Preencha os dados: Bandeira, Número (parcial), Limite, Data de Vencimento
Importante: O número do cartão deve ser inserido parcialmente (ex: ****.1234) para segurança. O sistema não armazena dados sensíveis completos.

🔄 Sincronização Automática

Se desejar, configure a sincronização automática com seu banco (disponível para bancos parceiros). Isso mantém seu saldo atualizado em tempo real.

3. Registrando Movimentações Diárias

O registro de transações diárias é essencial para manter um controle financeiro preciso e eficaz.

📝 Tipos de Transações

Receitas: Entradas de dinheiro provenientes de salários, freelas, aluguéis, etc.

Despesas: Saídas de dinheiro para contas, compras, investimentos, etc.

Transferências: Movimentações entre suas próprias contas (não afetam o saldo total).

⚡ Formas de Registro

Método Rápido: Use o botão flutuante "+" na tela principal para registrar transações imediatas.

Formulário Detalhado: Acesse "Transações" > "Nova Transação" para registros mais complexos com categorias específicas.

Recurso Inteligente: O sistema aprende com seus padrões de gastos e sugere categorias automaticamente com base em descrições anteriores.

🏷️ Categorização Inteligente

Categorias pré-definidas incluem:

  • Alimentação
  • Moradia
  • Transporte
  • Educação
  • Lazer
  • Saúde
  • Investimentos

Você pode criar categorias personalizadas conforme suas necessidades específicas.

4. Planejamento Financeiro Avançado

O SmartFinance oferece ferramentas poderosas para planejar seu futuro financeiro com estratégias avançadas.

🎯 Definição de Metas Financeiras

Estabeleça objetivos claros com prazos definidos:

  • Compra de imóvel
  • Férias internacionais
  • Fundo de emergência
  • Aposentadoria antecipada

Para cada meta, defina:

- Valor total necessário
- Data de conclusão desejada
- Valor mensal a ser poupado
- Conta específica para acumular recursos

📊 Análise de Gastos Recorrentes

O sistema identifica automaticamente assinaturas, aluguéis e outros gastos fixos, permitindo:

  • Análise de custo-benefício de serviços
  • Identificação de gastos desnecessários
  • Planejamento de substituição ou cancelamento
Dica Estratégica: Configure alertas para quando determinadas categorias ultrapassarem percentuais predefinidos do seu orçamento mensal.

🔄 Orçamentos Mensais Dinâmicos

Crie orçamentos baseados em porcentagens de sua renda ou valores fixos. O sistema ajusta automaticamente conforme sua realidade financeira evolui.

5. Investimentos e Aplicações

O SmartFinance ajuda você a gerenciar e acompanhar todos os seus investimentos em um único local.

💼 Tipos de Investimentos Suportados

Renda Fixa: Tesouro Direto, CDB, LCI, LCA, Debêntures

Renda Variável: Ações, ETFs, Fundos Imobiliários

Internacional: Ações americanas, BDRs, Criptomoedas

Previdência Privada: PGBL, VGBL e planos de aposentadoria

📈 Acompanhamento em Tempo Real

Integre corretoras participantes para atualização automática de cotas e valores. Para investimentos manuais, utilize o sistema de importação de extratos.

Calculadora Financeira: Ferramentas integradas para simular rentabilidades, calcular IR, e projetar crescimento de carteiras ao longo do tempo.

📊 Relatórios de Performance

Gere relatórios detalhados com:

  • Retorno sobre o investimento (ROI)
  • Comparação com índices de mercado
  • Alavancagem patrimonial
  • Distribuição por classe de ativo
Atenção: Mantenha sempre seus dados de login de corretoras em local seguro. O SmartFinance não armazena credenciais de acesso externo.

6. Perguntas Frequentes (FAQ)

❓ Como posso recuperar minha senha?

Clique em "Esqueci minha senha" na tela de login e siga as instruções enviadas para seu email cadastrado. O processo leva até 5 minutos para ser concluído.

❓ Os dados estão seguros?

Sim, utilizamos criptografia AES-256 e armazenamento em nuvem protegida. Nenhum dado bancário ou pessoal sensível é compartilhado com terceiros.

❓ Posso usar em múltiplos dispositivos?

Totalmente compatível com smartphones, tablets e computadores. Sincronizamos todos os dados em tempo real entre dispositivos.

❓ Como faço backup dos meus dados?

Acesse "Configurações" > "Exportar Dados" para baixar um arquivo JSON com todas suas informações financeiras.

Recomendado: Exporte seus dados mensalmente para manter cópias de segurança locais.

❓ Há limite de transações?

Não há limite para o número de transações que você pode registrar. O plano gratuito suporta até 10.000 lançamentos.

❓ Como funciona a sincronização bancária?

Disponível para clientes premium, conectamos diretamente com mais de 20 instituições financeiras para importação automática de movimentações.

8. Como Importar Dados via Planilha CSV

📋 Formato Requerido

Para importar transações existentes de planilhas, utilize o formato CSV com as seguintes colunas obrigatórias:

data,descricao,valor,tipo,categoria,conta_origem,conta_destino
2024-01-15,"Supermercado Dia",-285.50,despesa,"Alimentação","Carteira","Nubank"
2024-01-20,"Salário Empresa X",5200.00,receita,"Salário","Itaú",""
2024-01-25,"Transferência Poupança",-1000.00,transferencia,"Investimentos","Corrente","Poupança"

⚙️ Processo de Importação

1. Acesse "Configurações" > "Importar Dados"

2. Selecione o arquivo CSV formatado corretamente

3. Mapeie as colunas do seu arquivo com os campos do sistema

4. Revise as transações antes da confirmação final

5. Confirme a importação e aguarde a validação

Importante: Valide cuidadosamente o formato de datas (AAAA-MM-DD) e separadores decimais (.) antes da importação para evitar erros.

🔄 Mapeamento Personalizado

Se sua planilha tem cabeçalhos diferentes, você pode mapear manualmente cada coluna durante o processo de importação. O sistema reconhece variações comuns como "data_lancamento", "valor_bruto", "tipo_movimento", etc.

✅ Validação Automática

O sistema verifica automaticamente:

  • Consistência de datas
  • Valores numéricos válidos
  • Tipos de transação permitidos
  • Existência de contas mapeadas
